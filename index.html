Gameplay Tips

  • Eat the colorful food items (red, orange, yellow, blue) to grow and score points
  • Each food item adds 1 point to your score
  • 4 food items are always on the board
  • Avoid hitting your own body in both modes
  • In Classic mode, avoid the walls
  • In Fun mode, use wall wrapping to your advantage

üöÄ Installation

Prerequisites

  • Python 3.7 or higher
  • pip (Python package installer)

Setup Steps

1. Clone the repository:

git clone https://github.com/anbie/snake-game.git
cd snake-game

2. Install dependencies:

pip install -r requirements.txt

3. Run the game:

python snake_game.py

üß™ Testing
